이 ZIP 안 파일을 project 폴더에 그대로 덮어쓰기/추가하면 됩니다.

포함 파일
- main_master_v3.py
- rank_card_v3.py
- news.py
- reels_packager.py

작동 방식
1. 속보 감지 시 → 즉시 이미지 업로드
2. 한국시간 오전 8시 / 오후 7시 → 카드 3장 생성
   - 뉴스 TOP5
   - 폴리마켓 TOP5
   - 시장 반응 TOP5
3. 카드 3장을 텔레그램으로 전송
4. 같은 시점에 아래 텍스트 묶음도 텔레그램으로 전송
   - 커버 문구
   - 피드 제목
   - 피드 캡션
   - 릴스 훅
   - 릴스 스크립트
   - 릴스 캡션
   - 스토리 문구
   - 해시태그
   - 첫 댓글

주의
- 릴스 mp4 생성은 이번 버전에서 제외
- 인스타 음악은 직접 넣으면 됨
- 피드 1장 커버는 직접 제작하면 됨
- 텔레그램 개인 운영방 또는 비공개 채널로 받는 방식 추천

Railway Start Command
